Browser Objects -> window, navigator, screen
window -> document, history, location

We would see very brief info about each. Read good enough details from w3schools.com

Navigator object contains info about browser.

Screen object contains info about user screen

History object contains info about browser history



Location object contains info or interact with URL

Window object points to current open window

Our code (variable, methods) ... becomes part of the Window object. Think of history, location, document object as window attributes.
